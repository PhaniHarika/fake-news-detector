# 📰 Fake News Detector

ML-powered fake news detection with Gemini AI explanations.

## 🔴 Live Demo
👉 [Click here](https://fake-news-detector-4lde832kskcv7xewn2ysvk.streamlit.app/)

## 🛠️ Tech Stack
TF-IDF • Logistic Regression • Gemini AI • Streamlit • Python

## ✨ Features
- 98.78% accuracy on 44,000+ articles
- Real vs Fake classification with confidence score
- Gemini AI explains WHY news is fake/real
